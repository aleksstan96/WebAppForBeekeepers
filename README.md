# Honeyland

## Overview

Welcome to **Honeyland**! This is a Spring Boot and Angular 10 web app that helps beekeepers manage their work and sell their products. In addition, there is also an E-Commerce app for buyers included.

## Features

- Apiary Management
- Scheduler
- Tools
- Web Shop
